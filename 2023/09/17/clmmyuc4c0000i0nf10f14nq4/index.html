前端常用库

总结了一些常用js开发库

gsap

强大的JavaScript工具集,支持自定义曲线的js动画。

Three.js

是最受欢迎的 3D WebGL 库之一，为无数 3D 体验提供支持。也是最好的 3D 库之一。

Echarts

一个基于 JavaScript 的开源可视化图表库。

wangEditor

开源Web富文本编辑器，开箱即用，配置简单。

Pikaday

这是一个轻量级且可定制的日期选择器库。支持多种日期格式和本地化。

Fuse

轻量级的模糊搜索库。

Hotkeys

为指定操作注册快捷键。

Split

用于可调整大小的的分割布局。

Tippyjs

根据自己的需求和创意创建更复杂的工具提示、弹出窗、下拉菜单等元素。

Zdog

它是一个伪3D引擎，它的几何体存在于3D空间中，但以平面形状的方式呈现。

vConsole

H5开发必备。

lodash

是一个模块化、高性能，多功能的 JavaScript 实用工具库。

dayjs

Day.js是一个极简的处理时间和日期的JS库,无需考虑h5中android和ios的兼容性问题。

qs

qs是一个轻量的 url 参数转换的 JavaScript 库

animate.css

跨浏览器的 css3 动画库，内置了很多典型的 css3 动画，兼容性好，使用方便。

animejs

功能强大的 Javascript 动画库。可以与CSS3属性、SVG、DOM元素、JS对象一起工作，制作出各种高性能、平滑过渡的动画效果。

particles

构建粒子动画效果的js库。
